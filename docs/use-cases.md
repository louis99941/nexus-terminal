# 使用场景

Nexus Terminal 适用于多种远程服务器管理场景，以下是典型用例。

## 运维管理

### 服务器批量管理

通过多标签页和批量命令功能，同时管理数十台服务器：

- 同时登录多台 Web 服务器检查负载
- 批量执行系统更新和安全补丁
- 统一配置 Nginx、MySQL 等服务
- 一键收集所有服务器的磁盘使用情况

### 故障排查

利用 AI 智能助手快速定位问题：

- 自然语言描述问题，AI 生成排查命令
- 实时查看多台服务器的日志输出
- 通过 SFTP 快速下载故障日志分析
- 会话挂起功能确保排查过程不中断

### 监控巡检

仪表盘实时掌握集群状态：

- CPU、内存、磁盘、网络流量一览
- Prometheus 指标集成现有监控体系
- 异常事件通过 Webhook/Telegram 实时告警

## NAS 管理

### 家庭 NAS

通过浏览器安全访问家庭 NAS 设备：

- 无需安装客户端，任何设备均可访问
- SFTP 双面板文件管理，轻松整理照片和文档
- 移动端适配，手机也能管理 NAS
- 2FA 认证保护家庭数据安全

### 企业 NAS

集中管理多台存储设备：

- 同时连接多台 NAS 进行文件同步
- 通过 RDP 访问 NAS 的图形管理界面
- 审计日志记录所有文件操作
- IP 白名单限制访问来源

## 开发调试

### 远程开发

在本地浏览器中获得完整的远程开发体验：

- SSH 终端运行开发服务器和调试器
- Monaco Editor 在线编辑代码
- SFTP 上传和下载项目文件
- 分屏同时查看代码和运行结果

### CI/CD 管理

管理和调试持续集成流水线：

- 快速登录构建服务器查看日志
- 批量清理构建缓存和临时文件
- AI 助手分析构建失败原因
- 快速命令一键执行常用运维操作

### 数据库管理

通过 SSH 隧道安全访问远程数据库：

- 连接跳板机后访问内网数据库
- 终端中运行 MySQL/PostgreSQL 命令行
- 通过 SFTP 传输数据库备份文件

## 团队协作

### 多人运维

团队共享远程访问平台：

- 统一的用户认证和权限管理
- 完整的操作审计日志
- 会话管理避免连接冲突
- 通知系统及时了解团队操作

### 教学演示

在线教学和培训场景：

- 屏幕共享服务器操作过程
- 学员通过 Web 浏览器跟随操作
- 无需安装任何客户端软件
- 支持多种操作系统和协议

## 场景对比

| 场景     | 推荐协议   | 核心功能          | 推荐配置  |
| -------- | ---------- | ----------------- | --------- |
| 运维管理 | SSH + SFTP | 批量命令、AI 助手 | 2 核 2GB+ |
| NAS 管理 | SSH + SFTP | 文件管理、移动端  | 1 核 1GB+ |
| 开发调试 | SSH + RDP  | 终端、远程桌面    | 2 核 4GB+ |
| 团队协作 | 全协议     | 审计日志、通知    | 4 核 8GB+ |
| 教学演示 | SSH + RDP  | 屏幕共享、多标签  | 2 核 4GB+ |
